ConfigFile object

Atttribute Value Description
Name String The service name.
Filename String The absolute path of the configuration file.
Owner String The user and group of the configuration file's owner inside containers, in a format accepted by the chown command. For example, for a file owned by user and group root, root:root or 0:0.
Permissions String The file mode bits for the configuration file inside containers, in a format accepted by the chmod command. For example, for read, write, and execute permission for the owner and read permission for group and others, go+r,u+rwx or 0744.
Content String The content of the configuration file template, as a single JSON string.
