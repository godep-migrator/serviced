Go templates in JSON values

Most values of ServiceDefinition objects may include Go template expressions. For more information, refer to the Go template package.

Control Center provides the following, additional functions for those expressions.

  • bytesToMB function
    Divide by 1024.
  • child function
    Use the context of a child service.
  • each function
    Iterate over items.
  • parent function
    Use the context of the parent service.
  • percentScale function
    Multiply an item by a percentage.
  • plus function
    Perform integer addition.
