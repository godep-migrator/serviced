bytesToMB function

Divide by 1024.

Syntax: bytesToMB Item

Example: {{bytesToMB .RAMCommitment 0.8 | bytesToMB}}

If the value of RAMCommitment is 2147483648 bytes, the preceding template is replaced by 16777 (17179818 / 1024). The item to transform is rounded and the result is converted to an integer.
